Affiliated research at **CISPA Helmholtz Center for Information Security** focusing on optimizing **High Bandwidth Memory with Processing-In-Memory (HBM-PIM)** architectures to address memory bottlenecks in high-performance computing environments.

## Focus areas

- Characterizing performance bottlenecks using the **DAMOV** methodology on Google Cloud VMs
- Hardware-software co-design for memory-bound HPC workloads
- Fault-tolerant distributed systems and scalable simulation frameworks (BioDynaMo, CARTopiaX, NeuroDev)
- Memory security approaches including encryption as an architectural defense

This work connects systems research with practical performance engineering on cloud infrastructure.
